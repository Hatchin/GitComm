# Git Command Short Cheatsheet
1. Basic
   - `clone or download`
   git clone <repo url> <customer folder name>
   
   - `save for change`
   git commit  <path_to_file> -m "<commit message>"
   
   - `push to remote repository`
   git push origin master

2. Version Control
   - `get current working branch`
   git branch
   
   - `create new branch`
